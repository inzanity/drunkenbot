8.	Future plans and ideas

8.1	Future plans
There is no plans for developing full version of the game. Game will not be supported afterwards.

8.2	Additional features

Some features may be implemented in this version of the game. Following list is sorted by importance:
	- Multiplayer support
	- More available technology trees for player to use. Needs lots of time for balancing.
	- Cooperative mode for multiplayer game

We considered following features, but decided to drop them out:
	- Damage modelling for each body part of the mech
	- Joystick support
